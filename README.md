# Digital marketing agency — сайт про AI, контент и digital-упаковку для малого бизнеса с отдельным фокусом на бьюти-нишу.

Статический landing page для агентства, которое помогает салонам красоты, косметологам, мастерам и бьюти-студиям упаковывать соцсети, выпускать контент и получать больше записей.

## Что внутри

- адаптивный лендинг;
- блоки услуг и пакетов;
- AI Production для бьюти-соцсетей;
- кнопки Telegram на @DigitalWave_vl;
- без форм и онлайн-оплаты на сайте.

## Публикация

Сайт можно разместить через GitHub Pages. Загрузите `index.html`, `styles.css`, `script.js` и `README.md` в корень репозитория и включите Pages в настройках репозитория.
